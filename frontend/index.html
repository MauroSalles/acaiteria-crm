📖 Como Funciona — Guia Rápido

Use o menu superior para navegar entre as categorias do sistema. Veja abaixo o passo a passo:

🛒 Vendas

  1. No menu superior, clique em Vendas → Registrar Venda
  2. Selecione o cliente (deve ser cadastrado com consentimento LGPD)
  3. Adicione os produtos desejados e a quantidade
  4. Aplique desconto (%) ou taxa de serviço se necessário
  5. Escolha a forma de pagamento e clique em Finalizar Venda

👥 Clientes

  1. Clique em Clientes → Cadastrar Cliente para novos cadastros
  2. O cliente deve aceitar o consentimento LGPD para ser registrado
  3. Em Clientes → Lista de Clientes, busque, exporte ou gerencie todos os cadastros

📦 Produtos

  1. Clique em Produtos no menu para cadastrar, editar ou desativar itens
  2. Defina nome, preço, categoria e descrição

💰 Financeiro

  1. Financeiro → Relatórios mostra vendas do dia, clientes frequentes e produtos mais vendidos
  2. Financeiro → Fechamento de Caixa consolida as vendas por data e forma de pagamento

🔒 LGPD

A política de privacidade detalha quais dados são coletados e os direitos dos clientes. Todo cadastro exige consentimento.
